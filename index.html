Raspberry Pi Makerspace

Welcome to the home of simple Raspberry Pi based web automation!




To get started you will need a raspberry pi if you don't have one already. They are available from a variety of suppliers including buyapi.ca. Our base SD card is the perfect product to accompany your first Raspberry Pi thanks to it's hassle free approach to getting you sert up for coding.

All of our Micro SD cards come with Raspbian OS pre-installed. Our best selling product is our web automation optomized Micro SD card. This card comes with two python scripts already available; one base and another more complex web automation program. These programs are designed to allow for easy customization using python. This allows for more flexibility when it comes to meeting for your unique web automation needs. All of this is possible thanks to Selenium and Chromium Web Driver packages that our devellopers have preinstalled for easy access on our web automation SD card.

How to get automation programs up and running

  1. Get a raspberry pi with a wifi conncetion
  2. Purchase one of our preformated sd cards
  3. Boot your Pi from our SD card
  4. Try out our preinstalled programs (Further instructions are included as a text document with each product)
  5. By writing your own python programs or by customizing the pre-existing programs, you can write web automation scripts to complete repetitive daily tasks like attendance
